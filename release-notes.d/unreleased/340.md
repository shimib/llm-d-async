Breaking: the Helm chart is renamed from `async-processor` to `llm-d-async` and now publishes to `oci://ghcr.io/llm-d/charts/llm-d-async`. Rendered resources are named `llm-d-async-*` with `app.kubernetes.io/name=llm-d-async`; because a Deployment's selector is immutable, upgrading from an existing `async-processor` install requires uninstall + reinstall. The old `charts/async-processor` package remains available (no redirect) but receives no new releases.
